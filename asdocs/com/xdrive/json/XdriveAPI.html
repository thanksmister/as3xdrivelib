Classpublic class XdriveAPI

The XdriveAPI class abstracts the Xdrive JSON API found at http://dev.aol.com/xdrive.

XdriveAPI is the helper class for accessing the Open Xdrive API. It provides much of the client-side functionality for applications; for example, logging in and out, uploading files, sharing files, and basic file management. These functions build the requests with the appropriate JSON payload and decode the responses.

OPC_URLproperty 
public static var OPC_URL:String = "http://public.xdrive.com/"

The the URL to the open pubish channel server, the public Xdrive server for RSS feeds and assets. RSS feeds can be accessed from the OPC server when a folder or collection is published. Publishing a Folder or Collection will produce a RSS feed that can be consumed by a RSS reader or any other type of application capable of parcing XML. This server has an open cross domain policy file so Flash/Flex/AIR applications can consume resources without encountering security sandbox violation issue.

uploadZip()method 
public function uploadZip(folder:Folder, file:FileReference, parentid:String = ""):void

This method allows you to upload multiple files to a single destination Folder. The request consists of the json portion with a destFolder telling us where to put the files, and a zip file containing any number of files that will be put in the destination. The path structure of the zip file is maintained, any needed paths are created. Uploads a zipped local directory and recreates the directory structure in the users account. A blank parent id will upload the file to the root ("My Xdrive") directory.

Parameters
folder:Folder — Folder to upload
 
file:FileReference — FileReference that contains reference to the zip file and is used to handle events
 
parentid:String (default = "") — The id of the parent Folder to upload the zip file
