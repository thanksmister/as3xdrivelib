Property detail
MAX_FILE_SIZEproperty
public static var MAX_FILE_SIZE:Number = 50000000

Maximum allowed file size for video transencoding of uploaded files files. This is optional to use with the maxFileSize() function, and since its public, you can reset this value to any max file size you would like to use within the maxFileSize() funciton.
